# 价格趋势理论

> 公开材料结构化整理，非荐股。趋势管方向与级别；仓位与手法见 `prompts/writing-rules.md`。

---

## 1. 一句话

**趋势是价格连续创新高/新低形成的方向性结构；确认靠量能与相关指数/板块共振；在明确反向结构出现前，假定趋势延续——跟势，不测顶底。**

---

## 2. 理论谱系（从哪来）

| 层级 | 代表 | 贡献 |
|---|---|---|
| 源头 | **道氏理论**（Charles Dow → Hamilton → Rhea） | 趋势三分、指数互认、量能验证、趋势惯性 |
| 经典技术 | Edwards & Magee《Technical Analysis of Stock Trends》 | 趋势线、形态、突破在道氏上的图谱化 |
| 成本视角 | **均线理论** | 均线≈市场平均成本；排列与回踩描述趋势状态 |
| 系统化 | **趋势跟踪 / 海龟**（Donchian → Turtle Rules） | 突破进、反向突破出、波动率定仓，禁止预测 |

实务上说的「价格趋势理论」，通常 = **道氏的定义与确认 + 后来者的执行系统**，而不是单一本奇书。

来源摘要：[Investopedia Dow Theory](https://www.investopedia.com/terms/d/dowtheory.asp)｜[Wikipedia Dow theory](https://en.wikipedia.org/wiki/Dow_theory)｜[StockCharts ChartSchool](https://chartschool.stockcharts.com/table-of-contents/market-analysis/dow-theory)｜[海龟规则概要](https://www.turtletrader.com/rules/)

## 3. 道氏六大支柱（可执行改写）

古典表述略有版本差异，落地为六条：

### 3.1 价格反映一切（已贴现）

已知信息、预期、情绪，最终都挤进价格。含义：**读盘优先于读故事**；消息用来解释结构，不单独推翻结构。

### 3.2 市场有三种趋势（同时存在）

| 级别 | 常见时长（量级） | 角色 |
|---|---|---|
| 主要趋势 Primary | 数月～数年 | 牛/熊主方向，交易与持仓的「大局」 |
| 次级趋势 Secondary | 数周～数月 | 主升中的回调，或主跌中的反弹（常回撤主趋势波段的 1/3～2/3） |
| 短暂波动 Minor | 数日～数周 | 噪音；短线情绪博弈多发生在此层 |

**要点**：短线选手常在 Minor 层操作，但若与 Primary 逆行，胜率与赔率都会变差。

### 3.3 主要趋势常分三阶段（心理）

以牛市为例（熊市对称）：

1. **吸筹**：知情/敏锐资金进，舆论仍悲观，价格不显眼  
2. **公众参与**：趋势被确认，跟风与基本面改善共振，涨幅主体常在此  
3. **派发**：一致乐观、情绪高潮，聪明资金兑现  

与短线情绪周期「冰点→启动→发酵→高潮→退潮」同构，只是时间尺度不同。

### 3.4 指数（或相关品种）必须相互确认

道氏原用工业 vs 铁路（运输）：**同向创新高/新低才确认主趋势**。  
A股映射示例：

- 大盘：上证与深成/创业是否同向  
- 风格：权重 vs 成长是否一起走（背离=结构行情或假强）  
- 板块：龙头与中军/跟风是否共振（单股狂欢≠板块趋势）

### 3.5 成交量确认趋势

- 主升：上涨放量、回调缩量更健康  
- 主跌：下跌放量、反弹缩量更确认  
- 价涨量缩 / 价跌量缩：趋势可能乏力或仅为次级修复  

量是**确认项**，价格结构仍是第一决定因子（Hamilton 亦重视价优于量）。

### 3.6 趋势延续直至明确反转

**趋势在被证伪前有效。**  
反转要「清晰」：通常要结构破坏（如抬高的低点被跌破），而非单日长影线。代价是入场偏晚、离场偏钝——换来的是少被噪音清洗。

扩展实战规则（后人总结，非 Dow 原文，但常用）：

- **123 法则**：趋势终结三步确认（破前低/前高 → 次级反弹无力创不出新高/新低 → 再破近端结构）  
- **2B 法则**：假突破后迅速收回，常是反向信号  

参考：[EBC 道氏六原则与 123/2B](https://www.ebc.com/zh-cn/jinrong/302817.html)

## 4. 趋势的「操作定义」（画图用）

| 状态 | 价格结构 |
|---|---|
| 上升趋势 | **更高的高点 + 更高的低点**（HH + HL） |
| 下降趋势 | **更低的高点 + 更低的低点**（LH + LL） |
| 震荡/无趋势 | 高低点无递进，区间上下沿明确 |

判定习惯：

- 优先用**收盘价**判断是否破位（道氏重视收盘共识）  
- 次级回调未破前显著 HL → 主升未死  
- 反弹未破前显著 LH → 主跌未死  

这是所有趋势线、通道、波段标注的公共根基。

## 5. 均线：成本化的趋势滤波器

[均线理论](https://baike.baidu.com/item/%E5%9D%87%E7%BA%BF%E7%90%86%E8%AE%BA/1853202)把「价格序列」平滑为**市场平均成本轨迹**：

| 现象 | 解读 |
|---|---|
| 多头排列（短>中>长且向上） | 成本重心上移，趋势偏多 |
| 空头排列 | 成本重心下移，趋势偏空 |
| 回踩均线不破再起 | 次级回撤后主趋势延续的常见形态 |
| 均线缠绕走平 | 无趋势，突破系统易假信号 |

与道氏关系：均线不「创造」趋势，只是把 HH/HL 结构变得可量化、可程序化。

## 6. 趋势跟踪系统（海龟一脉）

哲学：**不预测趋势何时开始，等价格突破证明方向后跟随**；错了按波动止损，对了沿趋势持有/金字塔加仓。

经典规则骨架（Turtle）：

| 模块 | System 1（快） | System 2（慢） |
|---|---|---|
| 入场 | 突破近 20 日高/低 | 突破近 55 日高/低 |
| 出场 | 反向破 10 日极值 | 反向破 20 日极值 |
| 风控 | 波动率 N（ATR）定仓；常 2N 止损 | 同左 |
| 加仓 | 顺势每 ½N 可加单位，上限单位数 | 同左 |

局限（已知）：

- **震荡市**假突破多、连续止损  
- 入场偏晚、回吐利润一段是设计代价  
- 需要纪律与仓位算法，不是「看见突破就满仓」

来源：[TurtleTrader 规则说明](https://www.turtletrader.com/rules/)｜[海龟法则中文概要](https://incosmos.vision/education/5369/)

## 7. 与盘面复盘 / 短线心法如何对接

| 价格趋势理论 | 短线执行心法（writing-rules） |
|---|---|
| Primary / Secondary / Minor | 情绪周期嵌在短暂～次级波动层 |
| 量价确认 | 封板质量、换手、板块资金共振 |
| 指数互认 | 龙头带中军/跟风，忌「龙头孤岛」 |
| 趋势未死前跟随 | 只做主升与惯性；不做逆势抄底当主策略 |
| 突破跟随 | 「趋势加速」「破关键位放量」类买点 |
| 震荡退出/空仓 | 退潮、题材过散时的空仓合法 |

**实用分拆**：

1. **趋势层（周/日）**：是否 HH+HL？是否与指数/板块互认？量是否确认？  
2. **情绪层（日）**：冰点还是发酵？允许哪些手法？（见 `prompts/writing-rules.md` §2）  
3. **执行层**：模式内买点；结构破坏或逻辑证伪则出。

## 8. A股落地检查清单

下单/持仓前：

- [ ] 本标的日线是 HH+HL、LH+LL 还是震荡？  
- [ ] 相关指数或同板块是否同向确认？  
- [ ] 上涨是否相对放量、回调是否相对缩量？  
- [ ] 当前动作是顺着 Primary，还是逆着 Primary 做 Minor？  
- [ ] 若只是突破信号：是否接受「偏晚进、假突破止损」的交易成本？  
- [ ] 情绪阶段是否允许该周期的仓位？（对照游资心法 §2）

## 9. 常见误区

| 误区 | 纠正 |
|---|---|
| 趋势=永远上涨 | 下降也是趋势；「跟势」含空头侧或空仓 |
| 一根大阳线=转势 | 需结构（低点抬高/破前高等）确认 |
| 指标领先价格 | 趋势论以价格结构为主，指标是滤波器 |
| 必须买在起点 | 经典目标是吃主段，不是抄绝对底 |
| 短线情绪票等于趋势票 | 连板高潮常落在派发/Minor 噪声带，需升维看日线是否真转为上升结构 |

## 10. 最小可操作框架（研究结论）

```text
1. 定义：HH+HL / LH+LL / 震荡
2. 确认：量能 + 指数或板块互认
3. 策略：趋势在 → 顺势；震荡或证伪 → 轻仓/空仓/破位系统
4. 执行：突破或回踩成本区（均线）二选一，勿混用预测抄底
5. 离场：结构破坏（关键 HL/LH 被破）或系统出场规则触发
6. 叠加：用情绪周期调节仓位与手法，而不是推翻价格结构
```

**研究结论**：价格趋势理论的内核不是指标花园，而是「结构定义 + 确认 + 惯性假设 + 系统跟随」。短线要提高胜率，应先回答「我在主趋势的哪一层噪音里交易」，再用游资心法管情绪节奏。
