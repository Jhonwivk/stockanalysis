# 短线执行心法（盘面复盘配套）

> 版本：2026-07-14  
> 归属：`market-analysis` 技能内置模块  
> 性质：短线共性框架凝练，用于把量化 Regime/情绪映射为仓位与手法；不作投资建议。

---

## 0. 总纲

**只赚主线龙头、情绪共振、模式内的钱；行情差时空仓，逻辑证伪即出。**

六字诀：**势 · 龙 · 纯 · 空 · 证 · 复**

| 字 | 含义 |
|---|---|
| 势 | 先判情绪周期与赚钱效应，再选手法 |
| 龙 | 只做主线最强辨识度标的，不做杂毛 |
| 纯 | 模式少而精，模式外一律不做 |
| 空 | 空仓是仓位的一部分，不是偷懒 |
| 证 | 证伪就卖，不跟成本谈恋爱 |
| 复 | 每日复盘闭环，用结果改规则 |

---

## 1. 十条铁律

1. **先状态，后手法**：退潮/冰点禁止按高潮手法打满仓板。
2. **先主线，后个股**：无**确立**主线或题材过散，宁可空仓。主线必须用**近几日**连续性验证；**切换首日的新热点只能叫候选，不叫主线**。禁止把当日脉冲（含白酒/影视一日强）抬成主线。旧主线退潮后，先空旧、再轻试新，等次日确认再加码。
3. **先龙头，后跟风**：宁做龙尾，不做最强之外的跟风杂毛。
4. **二板（或身位）确认优先于一板幻想**：一板看故事，二板验资金共识。
5. **买在分歧，卖在一致**：高潮一致是兑现区，不是加仓区。
6. **只做主升与惯性**：不做反抽、不做「逆袭抄底」当主策略。
7. **新题材优先于旧题材**：资金最锐利的一股凝固在新叙事上。
8. **模式外 = 禁区**：看着眼馋但不在系统内的票，当作不存在。
9. **连续亏损强制休息**：连亏 2 笔则减仓/空仓复盘。
10. **盈亏交给概率，对错交给规则**：做对仍可能亏；做错必纠。

## 2. 情绪阶段 → 仓位与手法（写 §九 必用）

```text
冰点 → 启动 → 发酵 → 高潮 → 分歧 → 退潮 →（再）冰点
```

| 阶段 | 盘面近似特征 | 仓位基调 | 允许手法 | 禁止 |
|---|---|---|---|---|
| 冰点 | 跌停多、昨涨停大面、高度塌 | 空～极低 | 极小仓试新方向/观望 | 追高打板、幻想 V 必成 |
| 启动 | 跌停收敛、首板质量升、新题材露头 | 试错 1–3 成 | 低位启动首板、试错新龙 | 高位接力 |
| 发酵 | 涨停回升、晋级改善、主线清晰 | 半仓抬升 | **二板接力**、主线中军 | 追杂毛、多题材分散 |
| 高潮 | 加速缩量、一致性强、人气爆 | 持有/分批兑现 | 持龙；弱转强慎加 | **新开高位重仓** |
| 分歧 | 炸板升、高低切换 | 降仓观察 | 真龙**回封或反包** | 跟风首阴死扛 |
| 退潮 | 亏钱效应主导、高位补跌 | 空～防守 | 空仓为主 | 抄底退潮高位链 |

量化对照（优先用当日 JSON）：

- `emotion.score` / `stage`、涨停跌停比、炸板率  
- `yesterday_zt_today`：均收益、红盘率、晋级率、大面数  
- `zt_ladder`：最高板与梯队；主线持续性  

**口诀**：情绪决定仓位，主线决定方向，身位决定标的，盘口决定时点。

## 3. 主线 × 龙头

### 3.1 自上而下

```text
指数环境 → 全场情绪 → 最热题材（资金+涨停共振）→ 板块联动 → 龙头/中军 → 分时盘口
```

主线尽量满足：叙事清晰；涨停家数与板块涨幅/流入同向；能带跟风而非单股狂欢；近 2–3 日未被证伪。

题材过多时只跟当日「精神领袖」，其余当不存在。

### 3.2 龙头身位（满足越多越好）

| 维度 | 强信号 |
|---|---|
| 高度/身位 | 同题材最高板或最先突破关键位 |
| 时间 | 更早、更强封板 |
| 量价 | 充分换手；忌无量吃独食 |
| 带动 | 能带跟风或中军 |
| 抗跌 | 大盘弱时相对抗跌 |
| 辨识度 | 市场共识明确 |

不做：无主线杂毛、退市整理股当空间龙、退潮补跌反抽、成本焦虑单。

## 4. 节奏与手法箱

| 动作 | 含义 |
|---|---|
| 买分歧 | 抛压释放仍能回封/站稳 |
| 卖一致 | 全民共识、缩量加速或高潮末端 |
| 弱转强 | 昨弱今超预期（仅高辨识度） |
| 切换 | 新精神领袖出生则弃旧 |

| 手法 | 更适配阶段 |
|---|---|
| 质量首板 | 启动、冰点末试错 |
| 二板接力 | 发酵 |
| 持有兑现 | 高潮早期 |
| 反包/回封 | 分歧日真龙 |
| 低吸/趋势加速 | 趋势清晰 |
| 空仓 | 退潮或无模式机会（默认选项之一） |

同期主用 1–2 个手法，忌模式外乱做。

## 4b. 环境暗流（minor currents）→ 操作

主线三行之外，盘面常有 **支线/脉冲/残留/跟风** 暗流。暗流变化要翻译成动作，而不是改写主线标签。

| 暗流层 | 典型 | 仓位态度 |
|---|---|---|
| 退潮主流 | 昨最强今大面/跌停 | **空**（不接刀、不抄跌停反弹） |
| 主观察候选 | 切换后承接、近窗有痕迹 | **试**（≤总仓试错上沿） |
| 脉冲 | 白酒/影视等一日强 | **旁观**（不配主仓） |
| 残留 | 上一轮高位余热 | **不接力** |
| 跟风噪音 | 无叙事涨停 | **忽略** |

**环境变了怎么办（最小动作集）**：

| 变化 | 动作 |
|---|---|
| 退潮加深 | 保持空旧；打断抄弹性 |
| 候选内部分歧（高位核、中军在） | 减弹性，留或不留中军观察 |
| 候选升温（晋级/回流确认） | 提到试错仓上沿，标的收敛到 1 只逻辑 |
| 脉冲抢戏（酒影更强药转弱） | 当干扰，不切换信仰去追脉冲 |
| 竞价假切换（旧方向弱势反包） | 等确认，不在竞价定方向 |

纪律：**暗流再多，不增加「同时交易的方向数」**——同时最多 1 条候选。

## 5. 仓位与证伪

- 仓位跟随情绪；龙头可重、跟风必轻；试错→确认→加仓，证伪立刻砍  
- **检验句**：若此刻空仓，还会不会买？——不会则卖  
- 离场看**逻辑证伪**（主线死、带动失效、结构破、情绪退潮、预期差反向），不跟成本谈判  
- 单笔证伪当日本日或次日处理；连亏/回撤超阈值 → 强制空仓复盘  

---

## 6. 写入报告时的强制映射

阅读顺序：**摘要 → 策略卡 → 明日计划(§二) → 今日执行(§三) → 论据**。

1. 判定情绪阶段 → 心法仓位/手法/禁做（写入 §一、§三）  
2. **先写 §二 明日多预案**（≥4 枝），再写分析论据  
3. §三：暗流表 + 触发表；同时最多 1 条候选  
4. §五：`--days 7` 主线演进 + 切换路径；三行标签  
5. 退潮信号：跌停↑ + 昨涨停转亏 + 高度塌  

勿罗列席位/流派。
